HEP Compliance Review

Review compliance of all event applications according to the UTM Student Activity Guidebook.

Compliance Criteria (Guidebook Section 3.2)
✅ Must Pass
  • Advisor/Principal approval obtained
  • Multiracial participation (except religious)
  • Submitted ≥ 14 working days before event
  • Previous event report has been submitted
⚠️ Semester Limits
  • Maximum 4 activities per semester per society
  • Events during exam week — requires special permission from TNC (HEMA)
  • Events must end before 12:00 midnight
  • External sponsors: submitted ≥ 30 working days
